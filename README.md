# Schezwan Story 🍜

A modern and responsive Indo-Chinese restaurant website designed to showcase delicious dishes, special offers, and an engaging online food experience with clean UI and smooth navigation.

## 🌐 Live Demo

https://schezwanstory-indo-chinese.netlify.app/

## ✨ Features

- Modern restaurant landing page
- Fully responsive design
- Interactive food sections
- Smooth scrolling navigation
- Clean and attractive UI/UX
- Fast-loading frontend website
- Mobile-friendly layout

## 🛠️ Tech Stack

- CSS3

- Responsive Web Design
